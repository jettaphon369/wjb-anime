แก้ไข:
- นำฐานดำที่ติดใน GLB ออกก่อนคำนวณตำแหน่งเท้า
- คำนวณขอบเขตโมเดลใหม่จากตัวละครจริง

- ยกเลิกแอนิเมชันเด้งขึ้นลงของตัวละครที่วางบนสนาม
- ยังคงการขยับหายใจแบบเบา ๆ โดยไม่ทำให้ลอย
- เพิ่ม cache version เป็น game.js?v=4.5
