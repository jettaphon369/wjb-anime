Pet House Defense v2.5 — Seven 3D Heroes

ตัวละครในสนามต่อสู้ถูกแทนด้วยไฟล์ GLB ทั้ง 7 ตัวแล้ว:
Brisky, Mio, Gabby, Wind, Titan, Rian และ Kaisen

ภาพ PNG เดิมยังใช้เฉพาะการ์ดตัวละครและหน้าเมนู เพื่อให้โหลดเร็วและอ่านง่าย
เกมต้องเชื่อมต่ออินเทอร์เน็ตเพื่อโหลด Three.js และ GLTFLoader จาก CDN
ควรเปิดผ่าน GitHub Pages หรือเว็บเซิร์ฟเวอร์ ไม่ควรเปิด index.html แบบ file:// โดยตรง
