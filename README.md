# WJB: Kingdom Defense

เกมเว็บ Tower Defense ต้นแบบสำหรับ GitHub Pages

## วิธีติดตั้งบน GitHub Pages
1. แตกไฟล์ ZIP
2. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดเข้า repository `wjb-anime`
3. ไปที่ Settings → Pages
4. Source: Deploy from a branch
5. Branch: `main` และ Folder: `/ (root)`
6. Save

เปิดเว็บที่ `https://jettaphon369.github.io/wjb-anime/`

## ระบบที่มีในเวอร์ชันนี้
- กระดาน 5 × 9 ช่อง
- ฮีโร่ 7 ตัว พร้อมความสามารถต่างกัน
- พลังงานสำหรับวางยูนิต
- ศัตรูหลายระดับและบอส
- 5 ระลอก พร้อมชนะ/แพ้
- ปุ่มหยุดเกมและปรับความเร็ว
- รองรับมือถือและคอมพิวเตอร์

โปรเจกต์นี้ใช้ HTML/CSS/JavaScript ล้วน ไม่ต้องติดตั้งแพ็กเกจเพิ่ม
