# Pet House Defense WebGL 3D

เว็บเกม 3D/WebGL สำหรับฝากบน GitHub Pages

## วิธีลง GitHub Pages

1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้เข้า Repository
3. ไปที่ Settings > Pages
4. เลือก Source: Deploy from a branch
5. เลือก Branch: main และ Folder: /root
6. กด Save
7. รอประมาณ 1-3 นาที แล้วเปิดลิงก์ GitHub Pages

## ไฟล์สำคัญ

- `index.html` = ตัวเกม
- `.nojekyll` = ช่วยให้ GitHub Pages เสิร์ฟไฟล์ตรง ๆ
- `README.md` = คำอธิบายโปรเจกต์

หมายเหตุ: เกมนี้โหลด Three.js ผ่าน CDN จึงต้องใช้อินเทอร์เน็ตตอนเปิดเล่น
